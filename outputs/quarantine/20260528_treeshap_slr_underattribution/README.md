# Quarantine — TreeSHAP/model-free-hybrid H-S outputs (pre Group-Sobol)

**Date quarantined:** 2026-05-28
**Bug tag:** treeshap_slr_underattribution

## Which bug applies

The SLR Hawkins-Sutton variance-attribution figures produced by the TreeSHAP
pipeline (`shapley_hawkins_sutton.py`) and the SHAP+model-free hybrid
(`hybrid_hs_slr_unified.py` + `render_hybrid_tipping_split.py`) **mis-attribute
the SLR variance**:

1. **TreeSHAP under-attributes the emissions axis ~5×** (8.6% vs ~33% at 2150)
   because the RFF cumulative-emissions features are highly collinear; TreeSHAP
   splits their joint contribution arbitrarily, so summed per-feature SHAP
   variance << the true group contribution.
2. **The model-free within-cell V_BRICK over-attributes BRICK ~5×** (40% vs
   ~12-17% at 2150) because it conflates the BRICK main effect with
   post×cfg interactions.

Replaced by **Group-Sobol** (`group_sobol_hs.py`): Saltelli pick-and-freeze with
empirical Wong-weighted cell sampling, grouped first-order indices for
emissions / climate / brick, normalized to the surrogate-explained variance,
with a real interactions wedge (aggregate ST − S₁) and the AIS-tipping wedge
retained. GMST internal stays the surrogate OOF residual; SLR internal stays the
model-free seed-augmentation.

Caveat carried forward: the SLR surrogate OOF R² ceilings at ~0.71 (2150) —
irreducible across 6 model families and two feature sets (the cfg/post→SLR
response surface is under-determined by 10k cells). The normalized figure drops
this model-unresolved ~28% by construction; see the companion scoping memo
`notes/` for how large a new ensemble would need to be.
